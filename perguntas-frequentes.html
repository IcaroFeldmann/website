Perguntas Frequentes

Como funciona o aplicativo?

O aplicativo funciona totalmente offline, sem conexão com a internet.
Ao executar o aplicativo pela primeira vez, será solicitado a criação de usuário e senha. Isto serve para adicionar uma camada extra de segurança, para que o usuário não deixe seus dados expostos, caso outra pessoa eventualmente utilize seu computador.

Somente o usuário tem acesso aos seus próprios dados, pois todos os dados ficam armazenados apenas no seu computador, incluindo nome de usuário e senha, visto que o aplicativo funciona totalmente offline, sem conexão com a internet.

Os dados são armazenados em um arquivo chamado "BancodeDados.db", localizado em uma pasta também chamada "Banco de Dados", dentro da pasta "Documentos" do computador. Este arquivo é criado junto com a pasta "Banco de Dados" quando o usuário cria usuário e senha, logo ao executar o aplicativo pela primeira vez. Caso você necessite ou queira restaurar o aplicativo e apagar todos os dados inseridos nele, basta excluir a pasta "Banco de Dados". Nesse caso, quando você executar o aplicativo, será solicitado novamente a criação de usuário e senha, visto que a restauração exclui inclusive o usuário e senha.

É recomendavél que o usuário faça backps manuais e exportações CSV regularmente para evitar eventuais perdas de seus dados.

O aplicativo está disponível para quais sistemas operacionais?

O aplicativo está disponível somente para sistemas Windows 10 ou superior.

O aplicativo necessita de conexão com a internet?

Não, pois o aplicativo funciona totalmente offline e os dados são armazenados no computador do usuário.

Onde meus dados são armazenados?

Os dados do usuário são armazenados localmente no seu próprio computador, e somente o usuário tem acesso a seus dados. Estes dados ficam armazenados na pasta "Banco de Dados", dentro da pasta "Documentos" do computador.

Posso fazer login em outro computador?

Não, pois o aplicativo funciona totalmente offline com um banco de dados embutido no computador do usuário. Ou seja, somente o usuário tem acesso a seus próprios dados, que ficam armazenados localmente. Contudo, é possível transferir os dados de um computador para outro para sincronizar os dados, mas será necessário fazer isso manualmente.

Você pode exportar backups manualmente e também arquivos CSV, passá-los para um pen drive ou copiá-los para serviços de nuvem como Google Drive ou One Drive, para posteriormente copiar ou baixar estes arquivos para o computador de destino e sincronizar os dados ao clicar em "Importar Backup" ou "Importação CSV" no aplicativo "IF Gestor Financeiro".

Obs: Antes de importar backups ou arquivos csv, certifique-se de que o arquivo seja o mais recente para evitar a perda, a inconsistência e a duplicação de seus dados.

É possível utilizar o aplicativo em um Smartphone ou Iphone?

Não, o aplicativo está disponível somente para sistemas operacionais Windows 10 ou superior.

É possível acessar o aplicativo a partir de um navegador?

Não, pois o aplicativo funciona localmente, necessitando ser instalado no computador do usuário.
