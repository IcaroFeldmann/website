Solução de Problemas

Por que ocorre um erro ao tentar fazer login?

Ao cadastrar o usuário e senha, é necessário ficar atento para não deixar espaços em branco, pois esse erro ocorre devido a espaços em branco inseridos pelo usuário nos referidos campos ou quando o usuário esquece seu nome de usuário ou senha.

Solução:

Se você ainda não tem nenhum lançamento inserido no aplicativo, basta ir em "Computador", em seguida "Documentos", onde aparecerá uma pasta chamada "Banco de Dados". Basta excluir a pasta "Banco de Dados", que o aplicativo será restaurado e todos os dados inseridos serão apagados. Após isso, você deve iniciar novamente o aplicativo IF Gestor Financeiro, e será solicitado a criação de usuário e senha.

Obs: Cuidado para não deixar espaços em branco.

Esqueci minha senha, é possível recuperá-la?

Você deve anotar o nome de usuário e senha na primeira vez em que executar o aplicativo, o qual mostra uma mensagem sobre.

Não é possível recuperar a senha, visto que o aplicativo funciona totalmente offline e somente o próprio usuário tem acesso a seus dados, que ficam armazenados no seu próprio computador. A senha do usuário é gerada através de salt hash. Portanto, não há como recuperar a senha se o usuário esquecê-la.

Solução:

Neste caso, será necessário excluir a pasta "Banco de Dados", localizada na pasta "Documentos" do computador, para que o aplicativo tenha seu banco de dados restaurado.

Obs: Isto apagará todos os dados do usuário e será necessário criar novamente usuário e senha. Por isso, é recomendavél que o usuário exporte os dados em arquivos CSV e faça backups manuais periodicamente para evitar que seus dados sejam eventualmente perdidos ou corrompidos.

Assim como a senha, caso seja esquecido o nome de usuário, não será possível recuperá-lo, visto que o aplicativo funciona totalmente offline e somente o próprio usuário tem acesso a seus dados.

Por que os dados da tabela estão ilegíveis?

Isto pode ocorrer se o aplicativo for encerrado de maneira inesperada, o que pode interferir no processo de criptografa e descriptografia dos dados.

Solução:

Neste caso, será necessário excluir a pasta "Banco de Dados", localizada na pasta "Documentos" do computador, para que o aplicativo tenha seu banco de dados restaurado.

Obs: Isto apagará todos os dados do usuário e será necessário criar novamente usuário e senha.
Por isto, é recomendavél que usuário exporte os dados em arquivos CSV e faça backups manuais periodicamente para evitar que seus dados sejam eventualmente perdidos ou corrompidos.

Como restauro o aplicativo?

Para restaurar o aplicativo, você deve excluir a pasta "Banco de Dados", localizada na pasta "Documentos" do computador.

Obs: Isto excluirá todos os dados do banco de dados, incluindo usuário e senha.

Desinstalei e reinstalei o aplicativo. Por que segue sendo exibida a mensagem de erro ao tentar logar?

Não basta apenas desinstalar e instalar o aplicativo para restaurar o banco de dados. Isto ocorre porque os dados do usuário ficam armazenados na pasta "Banco de Dados", dentro da pasta "Documentos" do computador", mesmo após a desinstalação do aplicativo.

Solução:

Para restaurar o aplicativo, você deve excluir a pasta "Banco de Dados".
Obs: Isto excluirá todos os dados do banco de dados, incluindo usuário e senha.

Por que são exibidas mensagens relacionadas a erros na criptografia dos dados?

Isto ocorre quando o aplicativo é encerrado de forma inesperada, afetando o banco de dados no processo de criptografar e descriptografar os dados.

Solução:

Primeiramente, exporte os dados em formato CSV. Em seguida, será necessário excluir a pasta "Banco de Dados", localizada na pasta "Documentos" do computador.

Será necessário criar novamente usuário e senha.
Após fazer login será necessário importar o arquivo CSV exportado em "Importação CSV".

Caso os dados estejam ilegíveis, é possível importar backups manuais ou arquivos CSV feitos pelo usuário, após a exclusão da pasta "Banco de Dados". Caso não haja nenhum backup manual ou arquivo CSV exportado pelo usuário, os dados serão perdidos nesse procedimento.

Obs: É recomendavél que o usuário faça backups manuais e exportações CSV periodicamente para evitar a perda dos dados.

Os dados não estão aparecendo na tabela, o que aconteceu?

Em casos raríssimos, os dados não aparecem na tabela devido a resolução incorreta da tela do monitor do usuário.

Solução:

Basta alterar a resolução da tela para que os dados sejam exibidos na tabela.

O aplicativo não abre e é exibida a seguinte mensagem: "A aplicação já está aberta". Como solucionar?

Quando o usuário tenta abrir o aplicativo mais de uma vez ou clica diversas vezes sobre o ícone do aplicativo é exibida a mensagem de que o aplicativo já está aberto. Isto ocorre para evitar que o banco de dados do usuário seja corrompido.

Solução:

Abra o aplicativo apenas uma vez e aguarde até que a tela de login seja exibida.

Obs: Não clique diversas vezes sobre icone do aplicativo.
